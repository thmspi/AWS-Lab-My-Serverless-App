# Snippets du lab

- `lambda/lambda_function.py` : code complet à coller dans l'éditeur Lambda.
- `lambda/events/` : événement de test API Gateway avec claims Cognito simulés.
- `policies/` : politique S3 publique minimale, politique CloudFront/OAC et référence IAM least-privilege.
- `api/openapi-reference.yaml` : contrat de l'API, non destiné à l'import pendant le parcours console.
- `dynamodb/item-example.json` : forme d'un enregistrement de score.

Tous les textes `__COMME_CECI__` sont des placeholders à remplacer. La sandbox AWS Academy utilise `LabRole` ; la politique Lambda dédiée montre la configuration attendue dans un compte non restreint.
